# Lommix ui

# Colors

## valid values:

-   `20px`: Val::Px(20.)
-   `20vw`: Val::Vw(20.)
-   `20%`: Val::Percent(20.)

## valid ui rects:

-   `20px`: Uirect::all(value)
-   `20px 10px`: Uirect::axis(value, value)
-   `20px 10p% 5vw 3px`: Uirect::new(value,value,value,value)

## valid colors:

-   `#FFF`
-   `#FFFA` with alpha
-   `#FFFFFF`
-   `#FFFFFFAA` with alpha
-   `rgb(1,1,0.5)`
-   `rgba(1,1,0.8,1)`
